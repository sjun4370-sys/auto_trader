# 数据库设计

## 表结构总览

本系统共 12 张数据库表，涵盖用户、账户、交易、策略、AI等功能模块。

---

## 1. users - 用户表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| username | String(50) | 用户名，唯一 |
| email | String(100) | 邮箱，唯一 |
| password_hash | String(255) | 密码哈希 |
| is_active | Boolean | 是否激活 |
| is_superuser | Boolean | 是否超级管理员 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

---

## 2. exchange_accounts - 交易所账户表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| user_id | Integer | 外键 -> users.id |
| exchange | String(20) | 交易所标识 (okx, binance等) |
| account_name | String(50) | 账户名称 |
| api_key | String(255) | API Key |
| api_secret | String(255) | API Secret |
| passphrase | String(255) | 密码短语 (OKX用) |
| is_testnet | Boolean | 是否测试网 |
| is_active | Boolean | 是否激活 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

---

## 3. positions - 持仓表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| user_id | Integer | 外键 -> users.id |
| account_id | Integer | 外键 -> exchange_accounts.id |
| symbol | String(20) | 交易对 (BTC/USDT) |
| side | String(10) | 方向 (long, short) |
| quantity | Float | 持仓数量 |
| entry_price | Float | 开仓价格 |
| current_price | Float | 当前价格 |
| leverage | Integer | 杠杆倍数 |
| unrealized_pnl | Float | 未实现盈亏 |
| realized_pnl | Float | 已实现盈亏 |
| stop_loss | Float | 止损价格 |
| take_profit | Float | 止盈价格 |
| status | String(20) | 状态 (open, closed, liquidated) |
| opened_at | DateTime | 开仓时间 |
| closed_at | DateTime | 平仓时间 |

---

## 4. orders - 订单表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| user_id | Integer | 外键 -> users.id |
| account_id | Integer | 外键 -> exchange_accounts.id |
| strategy_id | Integer | 外键 -> strategies.id (可选) |
| symbol | String(20) | 交易对 |
| side | String(10) | 方向 (buy, sell) |
| order_type | String(20) | 订单类型 (market, limit) |
| quantity | Float | 数量 |
| price | Float | 价格 (限价单) |
| filled_price | Float | 成交价格 |
| status | String(20) | 状态 (pending, filled, cancelled, failed) |
| order_id | String(100) | 交易所订单ID |
| error_message | Text | 错误信息 |
| created_at | DateTime | 创建时间 |
| filled_at | DateTime | 成交时间 |

---

## 5. strategies - 策略表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| user_id | Integer | 外键 -> users.id |
| name | String(100) | 策略名称 |
| strategy_type | String(50) | 策略类型 (grid, dca, trend, ai) |
| config | JSON | 策略配置参数 |
| is_active | Boolean | 是否激活 |
| total_pnl | Float | 总盈亏 |
| win_rate | Float | 胜率 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

---

## 6. strategy_runs - 策略运行记录表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| strategy_id | Integer | 外键 -> strategies.id |
| status | String(20) | 运行状态 (pending, running, completed, stopped, error) |
| started_at | DateTime | 开始时间 |
| ended_at | DateTime | 结束时间 |
| last_executed_at | DateTime | 最后执行时间 |
| execution_count | Integer | 执行次数 |
| orders_count | Integer | 订单数量 |
| error_message | Text | 错误信息 |

---

## 7. trade_logs - 交易日志表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| user_id | Integer | 外键 -> users.id |
| symbol | String(20) | 交易对 |
| action | String(20) | 操作类型 (buy, sell, stop_loss, take_profit) |
| quantity | Float | 数量 |
| price | Float | 价格 |
| pnl | Float | 盈亏 |
| note | Text | 备注 |
| created_at | DateTime | 创建时间 |

---

## 8. system_status - 系统状态表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| status | String(20) | 系统状态 (running, stopped, error) |
| exchange_status | String(20) | 交易所连接状态 |
| last_heartbeat | DateTime | 最后心跳时间 |
| last_order_check | DateTime | 最后订单检查时间 |
| last_connection_check | DateTime | 最后连接检查时间 |
| pending_orders_count | Integer | 待处理订单数 |
| error_message | Text | 错误信息 |
| extra_data | JSON | 额外数据 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

---

## 9. optimization_history - 优化历史表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| user_id | Integer | 外键 -> users.id |
| strategy_id | Integer | 外键 -> strategies.id |
| analysis_period_start | DateTime | 分析开始时间 |
| analysis_period_end | DateTime | 分析结束时间 |
| performance_snapshot | JSON | 性能快照 |
| suggestions | JSON | 优化建议内容 |
| suggestions_count | Integer | 建议数量 |
| created_at | DateTime | 创建时间 |

---

## 10. ai_chat_history - AI对话历史表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| user_id | Integer | 外键 -> users.id |
| role | String(20) | 角色 (user, assistant) |
| content | Text | 对话内容 |
| created_at | DateTime | 创建时间 |

---

## 11. ai_signals - AI买卖信号表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| user_id | Integer | 外键 -> users.id |
| symbol | String(20) | 交易对 |
| signal_type | String(20) | 信号类型 (buy, sell, hold) |
| price | Float | 价格 |
| confidence | Float | 置信度 (0-1) |
| reason | Text | 信号原因 |
| status | String(20) | 状态 (pending, executed, expired, cancelled) |
| executed_at | DateTime | 执行时间 |
| created_at | DateTime | 创建时间 |

---

## 12. strategy_recommendations - 策略推荐表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增 |
| user_id | Integer | 外键 -> users.id |
| symbol | String(20) | 交易对 |
| recommended_strategy | String(50) | 推荐的策略类型 |
| market_condition | String(50) | 市场条件 |
| confidence | Float | 置信度 (0-1) |
| price | Float | 当前价格 |
| volatility | Float | 波动率 |
| volume | Float | 成交量 |
| volume_change | Float | 成交量变化率 |
| trend_strength | Float | 趋势强度 |
| reason | Text | 推荐原因 |
| is_applied | Boolean | 是否被应用 |
| created_at | DateTime | 创建时间 |

---

```
exchange_accounts (1) ─────< (N) positions
exchange_accounts (1) ─────< (N) orders

```
