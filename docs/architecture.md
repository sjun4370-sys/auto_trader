# 系统架构

## 整体架构

```
┌─────────────────────────────────────────┐
│              单体应用 (Python FastAPI)          │
├─────────────────────────────────────────┤
│  ┌─────────┐  ┌─────────┐  ┌────────┐  │
│  │ 行情采集 │  │ 策略引擎 │  │ 交易执行│  │
│  │ (异步)   │  │ (同步)   │  │ (同步)  │  │
│  └─────────┘  └─────────┘  └────────┘  │
│         │              │            │       │
│         └──────────────┼────────────┘       │
│                        ▼                    │
│                 ┌───────────┐            │
│                 │  SQLite   │            │
│                 │  (本地文件) │            │
│                 └───────────┘            │
└─────────────────────────────────────────┘
```

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 后端 | Python FastAPI | 高性能异步框架 |
| 数据库 | SQLite | 轻量级数据库 |
| 前端 | React + Ant Design | UI框架 |
| 交易所SDK | CCXT | 统一交易所接口 |
| 认证 | JWT | 无状态认证 |

## 模块设计

### 1. 认证模块 (auth)
- 用户注册/登录
- JWT Token 生成和验证
- 密码哈希存储

### 2. 账户模块 (accounts)
- 交易所账户绑定
- API 密钥管理
- 模拟/实盘切换

### 3. 行情模块 (market)
- 实时行情获取
- K线数据
- 订单簿数据

### 4. 交易模块 (trade)
- 下单/撤单
- 订单状态管理
- 成交记录

### 5. 持仓模块 (positions)
- 持仓查询
- 止盈止损设置
- 平仓操作

### 6. 策略模块 (strategies)
- 策略管理
- 策略启动/停止
- 策略配置

### 7. 风控模块 (risk)
- 仓位限制
- 止损机制
- 熔断保护

### 8. AI模块 (ai)
- 交易信号
- 市场分析
- 对话助手

### 9. 统计模块 (statistics)
- 盈亏统计
- 胜率分析
- 交易记录
